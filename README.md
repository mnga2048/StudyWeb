# StudyWeb · 专业课学习平台

> 面向电子、自动化、电气、通信专业学生与备考同学的**专业课一站式学习网站**。
> 应试（学习笔试）与工程（实战应用）双线并重，把分散的知识系统化、可视化、可交互地组织。
> 特别增设**机器人控制方向**课程（机器人学、电机拖动、嵌入式、传感器、现代控制等）。

[![线上访问](https://img.shields.io/badge/线上访问-Vercel-2563eb)](https://study-web-cckiller.vercel.app) [![License](https://img.shields.io/badge/license-MIT-green)](LICENSE)

每个知识点配备：正文讲解 + KaTeX 公式 + 交互图表 + 自测题。

## ✨ 核心特性

- **📚 18 大板块**：覆盖数学→电子→控制→嵌入式→机器人→计算机全栈知识
- **🧮 公式可视化**：KaTeX 渲染全部数学/控制公式，配套 9 个计算工具
- **🎮 交互图表**：伯德图、根轨迹、阶跃响应、RC 波形、排序算法等 10+ 种可交互图表
- **✏️ 自测练习**：615 道自测题，覆盖 128 个知识点，含解析与错题记录
- **📊 学习进度**：知识图谱、掌握度热力图、学习推荐、薄弱环节分析
- **🌙 深色模式** + 响应式布局，移动端友好
- **🔍 全文搜索**：支持标题、简介、标签检索，关键词高亮

## 🚀 本地预览

纯前端静态站点，无需构建。任选一种方式：

```bash
# 方式一：Python 内置服务器
python -m http.server 8000
# 浏览器访问 http://localhost:8000

# 方式二：直接双击 index.html（部分浏览器对本地文件有限制，推荐用服务器）

# 方式三：npx serve
npx serve .
```

## 🛠 技术栈

| 类别 | 技术 | 说明 |
|------|------|------|
| 样式 | Tailwind CSS (CDN) | 原子化布局 |
| 主题 | 原生 CSS 变量 | 学术蓝 + 深色模式 |
| 公式 | KaTeX + auto-render | 数学/控制公式渲染 |
| 图表 | ECharts 5 | 波形图、知识图谱、进度图 |
| 路由 | 原生 JS SPA | 单页应用，无框架 |
| 存储 | localStorage | 进度/收藏/答题/主题 |
| 构建 | 零构建 | 双击即用 |
