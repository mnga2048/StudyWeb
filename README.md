# StudyWeb · 专业课学习平台

> 面向电子、自动化、电气、通信专业学生与考研党的**专业课一站式学习网站**。
> 应试（考研笔试）与工程（实战应用）双线并重，把分散的知识系统化、可视化、可交互地组织。

[![线上访问](https://img.shields.io/badge/线上访问-Vercel-2563eb)](https://studyweb.vercel.app) [![License](https://img.shields.io/badge/license-MIT-green)](LICENSE)

## 📚 内容板块

| 板块 | 知识点数 | 侧重 |
|------|---------|------|
| 🔴 高等数学 | 16 | 考研应试 |
| 🟠 线性代数 | 10 | 考研应试 |

| 🟢 模拟电路 | 14 | 应试 + 工程 |
| 🟡 数字电路 | 14 | 应试 + 工程 |
| 🟣 自动控制原理 | 14 | 工程应用 |
| ⚫ 数据结构 | 16 | 408 + 工程面试 |

**合计 96 个知识点**，每个知识点配备：正文讲解 + KaTeX 公式 + 交互图表（计划中）+ 自测题（计划中）。

## ✨ 核心特性

- **🎯 双标签系统**：每个知识点标注"应试 / 工程"，可按模式筛选
- **🧮 公式可视化**：KaTeX 渲染全部数学/控制公式，配套矩阵计算器、拉氏变换查表
- **🎮 交互图表**：伯德图、根轨迹、卡诺图、运放电路、排序算法等可交互原理图（规划中）
- **✏️ 自测练习**：每节配自测题，含考研真题模块与错题记录
- **📊 学习进度**：localStorage 持久化，跟踪每个知识点的学习状态
- **🌙 深色模式** + 响应式布局，移动端友好
- **🔍 全文搜索**：支持标题、简介、标签检索，关键词高亮

## 🚀 本地预览

纯前端静态站点，无需构建。任选一种方式：

```bash
# 方式一：Python 内置服务器
python -m http.server 8000
# 浏览器访问 http://localhost:8000

# 方式二：直接双击 index.html（部分浏览器对本地文件有限制，推荐用服务器）

# 方式三：npx serve
npx serve .
```

## 🛠 技术栈

| 类别 | 技术 | 说明 |
|------|------|------|
| 样式 | Tailwind CSS (CDN) | 原子化布局 |
| 主题 | 原生 CSS 变量 | 学术蓝 + 深色模式 |
| 公式 | KaTeX + auto-render | 数学/控制公式渲染 |
| 图表 | ECharts 5 | 波形图、知识图谱、进度图 |
| 路由 | 原生 JS SPA | 单页应用，无框架 |
| 存储 | localStorage | 进度/收藏/答题/主题 |
| 构建 | 零构建 | 双击即用 |
